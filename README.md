# Sistema de Leilões - README

##  Sobre o Projeto
Este projeto visa informatizar as rotinas de uma casa de leilões. O objetivo é tornar o processo de leilão mais ágil.

##  Funcionalidades
- Cadastro de itens para leilão
- consultar produtos
-  lista de produtos
